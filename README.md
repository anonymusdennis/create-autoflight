# Create AutoFlight

NeoForge 1.21.1 companion mod for **Create Aeronautics** — gyroscope flight control, navigation, inertial thrusters, and thrust-vectoring gearboxes for Sable assemblies.

## Requirements

- Minecraft 1.21.1
- NeoForge 21.1+
- Create, Simulated, Sable, Create Aeronautics

## Build

Copy dependency JARs into `libs/` (see [libs/README.md](libs/README.md)), then:

```bash
./gradlew build
```
